# 🚀 Transfer-IQ: Player Market Value Intelligence

## 🌐 Live App  
### 👉 https://transfer-iq.streamlit.app/

---

## 📌 Project Overview
Transfer-IQ is a data-driven analytics project that predicts and analyzes football player market value by integrating multi-source data including performance metrics, injury history, and social media sentiment.

The goal is to build a robust ML-ready dataset and uncover key factors influencing player valuation.

---

## 🗂️ Data Sources
- **StatsBomb** — player performance data  
- **Transfermarkt** — market value data  
- **Social Media** — sentiment signals (VADER/TextBlob)  
- **Injury Records** — player availability and risk  

---

## ⚙️ Work Completed

### Week 1–2
- Multi-source data collection and alignment  
- Player identity standardization  
- Missing value handling and deduplication  
- Initial feature engineering  

### Week 3–4
- Sentiment analysis using VADER/TextBlob  
- Advanced performance efficiency metrics  
- Injury risk feature generation  
- Final modeling-ready dataset (~270 players)

---

## 🧠 Key Engineered Features
- Goal conversion rate  
- Minutes per goal  
- Performance index  
- Injury risk score  
- Sentiment score and polarity features
