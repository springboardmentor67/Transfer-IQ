# TransferIQ

Dynamic Player Transfer Value Prediction using AI and Multi-source Data.

TransferIQ is an AI-powered football analytics dashboard that predicts player market value using machine learning models such as LSTM and XGBoost.

The system analyzes player performance, injuries, and sentiment to generate accurate predictions and insights. It provides interactive visualizations like market trends, performance radar, and model comparisons using Streamlit and Plotly.

This project helps in making data-driven transfer decisions by combining analytics and predictive modeling in a user-friendly dashboard.
