# Week 2 — Data Preprocessing

This folder mirrors the Week 2 deliverable structure.

## What this week covers
- Cleaning (missing values, types, basic sanity checks)
- Feature engineering (simple aggregates/joins)
- Encoding & scaling

## Implementation mapping
The existing Week 2 pipeline lives in:
- `scripts/week2_data_processing.py`

Wrapper scripts are provided in `scripts/` here to match the requested structure.

## Run (from repo root)
```powershell
python scripts\week2_data_processing.py
```

## Outputs
Outputs are written to `data/processed/` by the pipeline.
Large generated artifacts are ignored by git.
